# 🎭 Театральный Генератор Постов

Локальное Flask-приложение для создания и экспорта анонсов театральных мероприятий в PDF.

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите приложение:
   ```bash
   python app.py
   ```

3. Откройте браузер и перейдите по адресу: `http://127.0.0.1:5000`

## Функционал

- Генерация текста поста на основе описания, жанра и настроения.
- Настройка "голоса" поста через `voice.md`.
- Реальный экспорт сгенерированного текста в PDF (с использованием reportlab).
- Тёмный интерфейс с адаптивной вёрсткой.

## Файлы

- `app.py` — серверная логика Flask.
- `templates/index.html` — интерфейс пользователя.
- `static/style.css` — стили (тёмная тема).
- `voice.md` — настройки тона и стиля постов.
- `requirements.txt` — зависимости Python.
